grid

/src/assets/css/templates/_grid.css
  • Grid

グリッドシステムです。カラムごとに任意の値を指定したい場合に使います。 カラムの指定はutility/_col.cssのスタイルを使います。

以下のような指定ができます。

  • .u-col-8of12 カラム数の指定。
  • .t-grid--medium ガターの指定。
  • .t-grid--reverse カラムの反転。
  • .t-grid--center カラムのセンタリング。
  • .t-grid--right カラムの右寄せ。

以下のような指定ができます。

  • 1/2から1/4までの均等なカラム指定。
  • ブレイクポイントなしから、-md、-lgでの指定。
