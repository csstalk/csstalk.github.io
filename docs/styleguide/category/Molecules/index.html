figure

/src/assets/css/molecules/_figure.css
  • Contents

キャプションをつけた画像コンポーネントです。キャプションは画像の横幅を基準に折り返されます。

Modifier

  • .m-figure--center - コンポーネント自体をセンタリングする
  • .m-figure__caption--center - キャプションをセンタリングする
figure.m-figure.m-figure--center
  img.m-figure__image(src="https://placehold.jp/300x200.png" alt="")
  figcaption.m-figure__caption キャプションキャプションキャプションキャプションキャプション
